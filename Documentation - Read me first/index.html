08) Create your keystore - top

8.1) Create your keystore file by running the below command in your terminal, make sure to point in your keytool.

On Mac/Linux, use the following command:
keytool -genkey -v -keystore ~/key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias key

On Windows, use the following command:
keytool -genkey -v -keystore c:\Users\USER_NAME\key.jks -storetype JKS -keyalg RSA -keysize 2048 -validity 10000 -alias key

Warning: Keep the key.properties file private; donâ€™t check it into public source control.

8.2) Reference the keystore from the app:
Open project {app dir}/android and create a file named -> key.properties and copy below information that contains a reference to your keystore inside it as shown on image below.



Copy the below code and replace with your information:

storePassword=password from previous step
keyPassword=password from previous step
keyAlias=key
storeFile=location of the key store file, such as /Users//key.jks

After following all steps above, now its time to run a command to buid your APK, but before starting, you need to clean debug cache. Open project {pp dir}/terminal and follow the below commands.
